## Example: Dual-Method Teaching

Here's a sample from my tutoring work showing both approaches to the same problem:

**Problem**: Show that if $f \in C^2$, $\lim_{xâ†’âˆž} f(x)$ exists and is finite, and $f''$ is bounded, then $\lim_{xâ†’âˆž} f'(x) = 0$.

**Street Fighter Approach**: "If $f$ is settling down to a horizontal asymptote, and $f''$ is bounded (so $f'$ can't wiggle too much), then $f'$ *must* go to zeroâ€”otherwise the function would keep rising or falling forever. The bounded second derivative prevents surprises."

**Good Boy Approach**: Construct the sequence $x_n = n\epsilon/M$, apply the Cauchy property to show $|f(x_{n+1}) - f(x_n)| \leq \epsilon^2/M$, use Mean Value Theorem to get $|f'(c_n)| \leq \epsilon$, then bound $|f'(u) - f'(c_n)| \leq M|u - c_n| \leq \epsilon$, concluding $|f'(u)| \leq 2\epsilon$ for arbitrary $\epsilon$.

**Both are correct. Both teach different skills. Students need both.**
